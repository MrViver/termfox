<b>-+=> How To Install:</b>

<b><i>From Termux(git clone) :</i></b>


<p>$ pkg install git</p>
<p>$ git clone https://github.com/MrViver/termfox</p>
<p>$ cd termfox</p>
<p>$ sh setup.sh</p>

<b><i>From Download(Download Zip) :</i></b>

download termfox-master.zip from github

<p>$ unzip termfox-master</p>
<p>$ cd termfox-master</p>
<p>$ sh setup.sh</p>
