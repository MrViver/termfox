<b>-+=> How To Install:</b>

<i>on termux:</i>

$ pkg install git
$ git clone https://github.com/MrViver/termfox
$ cd termfox
$ sh setup.sh

<i>on download:</i>

download termfox-master.zip from github
$ unzip termfox-master
$ cd termfox-master
$ sh setup.sh
